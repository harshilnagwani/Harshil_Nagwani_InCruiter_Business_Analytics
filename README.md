## 🛠 How to Compile

1. Open [Overleaf](https://www.overleaf.com)
2. Create a new blank project
3. Replace the default `main.tex` contents with the `.tex` file from this repo
4. Set compiler to **pdfLaTeX**
5. Hit **Recompile**
6. Download the PDF and upload it here as `Harshil_Nagwani_InCruiter_Business_Analytics.pdf`
